PALETTE COLORI UFFICIALE
------------------------
FONDO:           #F6F4EF (bianco sporco/sabbia)
BLU LO SQUALO:   #2F6F8F (asse principale - linee, hover, titoli)
TESTO:           #2E2E2E (principale) / #6B6B6B (secondario)

MACROCATEGORIE:
- EVENTI:              #D4A93C (giallo/oro - unico colore caldo)
- ALLOGGIO:            #E6BFC7 (rosa soft)
- ESCURSIONI:          #4F8C8C (verde acqua)
- SURFING TENERIFE:    #5FA6C6 (blu oceano)
- AGENCY:              #3E5C6E (blu scuro professionale)
- ALESSANDRO:          #6B8796 (grigio blu)

REGOLE:
- Un colore = una macro
- Sotto-voci = variazioni dello stesso colore
- Hover: +10% saturazione
- Active: +10% scuro
- Niente gradienti casuali / ombre colorate


FUNZIONALITA' IMPLEMENTATE
--------------------------

DESKTOP:
- Mind map interattiva con Tenerife al centro
- Apertura progressiva: Tenerife → 6 categorie → sotto-categorie
- Linee SVG dinamiche che collegano i nodi (calcolate in tempo reale)
- Disposizione a esagono delle 6 categorie attorno al centro
- Animazioni fluide con transizioni CSS
- Effetto hover sui nodi con glow colorato per categoria
- Pulsante "Indietro" per tornare allo stato precedente
- Supporto tasto ESC per navigare indietro
- Responsive scaling per schermi fino a 992px

MOBILE (< 768px):
- Mini-mappa semplificata con Tenerife e 6 categorie in esagono
- Tap su Tenerife → appaiono le 6 categorie con linee
- Tap su categoria → popup radiale con sotto-categorie disposte a cerchio
- Sfondo sfocato (blur) quando il popup e' aperto
- Chiusura popup toccando fuori o pulsante "X Chiudi"
- Elementi ottimizzati per touch (dimensioni generose)
- Supporto schermi piccoli (< 480px)

NOTE TECNICHE
-------------
- Le linee SVG sono calcolate dinamicamente con getBoundingClientRect()
- Aggiornamento linee ogni 50ms per seguire le animazioni
- Il popup radiale calcola le posizioni con trigonometria (seno/coseno)
- CSS variables per gestire la palette colori centralmente
- CSS media queries per responsive design
- Nessuna libreria esterna (vanilla JS)
